# Statement.md

Problem Statement: A digital library management system

Scope: We aims to help librarians and library-goers, streamlining their workflow and providing a digital solution for it.

Target users: Librarians and customers of libraries.

Features:

- Admin:
    1. Login into User Panel
    2. Modify User:
        I) Add User
        II) Delete User
        III) Update User
    3. Display Users
    4. Search Users
    5. Modify Book
        I) Add Book
        II) Delete Book
        III) Update Book
    6. Issue Book
    7. Return Book
    8. Change Admin
    9. Home
    10. Back
    11. Exit

- User:
    1. About Library
    2. News
    3. Wikipedia Articles
    4. Display Books
    5. Search Books
    6. Issued Books Details
    7. Notes
        - Modify Notes:
            - Add Notes
            - Delete Notes
            - Update Notes
        - Display Notes
        - Search Notes
    8. Home
    9. Back
    10. Exit
